Studybot - Chatbots pour l'éducation (EMLYON Business School)

Contexte

Création et déploiement de chatbots éducatifs utilisés par plus de 800 étudiants et collaborateurs de l'EMLYON Business School, avec une attention particulière portée à la conformité RGPD et à la scalabilité.

Défis

  • Développement d'assistants IA adaptés au contexte éducatif
  • Intégration avec les systèmes existants de l'école
  • Conformité RGPD stricte pour les données des étudiants
  • Mise en place d'une infrastructure évolutive pour supporter des pics d'utilisation

Solution technique

  • Déploiement sur Azure Kubernetes Service pour une haute disponibilité et scalabilité
  • Intégration d'Azure OpenAI avec des appels API conformes RGPD
  • Mise en place d'un système de journalisation sécurisé pour les interactions
  • Architecture microservices pour faciliter les mises à jour et la maintenance
  • Système d'authentification sécurisé avec Azure AD

Aspects RGPD

  • Anonymisation des données personnelles dans les requêtes API
  • Mise en place de politiques de rétention des données
  • Système de consentement explicite pour les utilisateurs
  • Documentation complète des traitements de données

Résultats

  • Adoption par plus de 800 utilisateurs
  • Amélioration de l'accès à l'information pour les étudiants
  • Réduction de la charge de travail pour le personnel administratif
  • Conformité totale avec les exigences RGPD
  • Infrastructure robuste capable de gérer des pics d'utilisation (jusqu'à 800 utilisateurs simultanés)

SynthéZ - Solution de transformation de documents pour EMLYON Business School

Contexte

Développement d'une solution innovante de transformation de documents pour EMLYON Business School, permettant de convertir des PDF longs en versions synthétiques, structurées et adaptées aux besoins pédagogiques.

Défis

  • Traitement intelligent de documents complexes et volumineux
  • Analyse et synthèse de documents pédagogiques
  • Création d'une interface utilisateur intuitive et efficace
  • Génération de contenu structuré et personnalisable

Solution

Développement d'une application web complète utilisant :

  • Azure OpenAI (GPT-4o) pour l'analyse et la génération de contenu structuré
  • DALL-E pour la création d'images de couverture personnalisées
  • React avec TypeScript pour le frontend (Tailwind CSS, shadcn/ui)
  • Azure App Service pour le backend (Node.js)
  • Azure Static Web Apps pour l'hébergement du frontend

Aspects techniques notables

  • Mise en place d'un système de proxy pour sécuriser les appels API
  • Configuration multi-environnements (local, dev, production)
  • Interface utilisateur intuitive avec glisser-déposer pour la réorganisation des sections
  • Génération automatique de PDF structurés avec mise en page professionnelle

Résultats

  • Gain de temps significatif pour les équipes pédagogiques dans la préparation des supports de cours
  • Amélioration de la qualité des synthèses grâce à une structure cohérente et personnalisable
  • Flexibilité d'utilisation avec support pour les documents PDF et les scripts vidéo
  • Expérience d'apprentissage améliorée pour les étudiants avec des documents synthétiques adaptés à leurs besoins

QuizForge - Outil de génération automatique de questionnaires

Contexte

Développement d'un outil de génération automatique de questionnaires à partir de documents pédagogiques pour les formateurs et enseignants, permettant de créer rapidement des évaluations de qualité basées sur le contenu de leurs supports de cours.

Défis

  • Extraction efficace du contenu de différents formats de documents (PDF, DOCX, ZIP)
  • Génération de questions pertinentes et variées grâce à l'IA
  • Standardisation des questions selon des référentiels pédagogiques
  • Export dans des formats compatibles avec les plateformes d'apprentissage (GIFT, Moodle XML)

Solution

Développement d'une application web full-stack utilisant Node.js/Express pour le backend et React pour le frontend, intégrant Azure OpenAI pour l'analyse de contenu et la génération de questions personnalisées.

Caractéristiques techniques

  • Backend en Node.js avec Express et SQLite pour le stockage
  • Intégration d'Azure OpenAI (GPT-4o) pour la génération intelligente de questions
  • Tests automatisés avec Jest (couverture de 27.58%)
  • Architecture modulaire avec séparation des services (parsing, IA, export)

Résultats

  • Réduction significative du temps de préparation des évaluations (de plusieurs heures à quelques minutes)
  • Génération de questions de différents niveaux de difficulté et formats
  • Possibilité d'aligner les questions sur des référentiels pédagogiques spécifiques
  • Export dans des formats standards compatibles avec les principales plateformes d'apprentissage

POC RAG - Retrieval Augmented Generation avec Flowise AI

Contexte

Développement d'un Proof of Concept (POC) utilisant la technique RAG (Retrieval Augmented Generation) avec Flowise AI pour améliorer la précision et la pertinence des réponses générées par des modèles d'IA générative.

Défis

  • Intégration de sources de données propriétaires dans un système de génération IA
  • Réduction des hallucinations et amélioration de la fiabilité des réponses
  • Optimisation des performances de recherche et de récupération d'informations
  • Création d'un système évolutif pouvant s'adapter à différents cas d'usage

Solution

Mise en place d'une architecture RAG complète utilisant Flowise AI, Azure OpenAI, des embeddings vectoriels et une base de données vectorielle pour enrichir les réponses du modèle avec des informations précises et à jour provenant de sources fiables.

Résultats

  • Amélioration de la précision des réponses de 78% à 96%
  • Réduction des hallucinations de 65%
  • Temps de réponse optimisé (moins de 2 secondes)
  • Preuve de concept validée et prête pour une implémentation à grande échelle
